# Models

Optional but recommended — the repo "may also include files for models used by 3D
printers, laser cutting machines and CNC machines" (rule 7).

- `/3d-printed` — STL, 3MF, or source CAD files for 3D-printed vehicle parts
- `/laser-cut` — DXF, SVG, or source files for laser-cut/CNC parts

Including these strengthens your "Reproducibility" score (Engineering Journal Criterion 5).
