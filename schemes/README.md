# Schemes & Diagrams

Put here anything visual that supports your README and Engineering Journal:

- `wiring-diagram.*` — full electrical wiring diagram (battery → regulators → controllers →
  motors/sensors)
- `flowchart.*` or `state-machine.*` — software control flow / state machine
- `chassis-cad.*` — CAD screenshots/exports of the mechanical design
- `power-budget.*` — table/chart of current draw per subsystem

These directly support scoring criteria 1, 2, and 3 in the Engineering Journal rubric
(Appendix C) — diagrams are explicitly called out as needed for a "Level 4 (Competent)" score
or higher.
